# Termux Tool

## USAGE:


1. Update Termux
```bash
pkg install && pkg update -y
```
2. Install git
```bash
pkg install git
```
3. Access permission
```bash
termux-setup-storage
```
4. Clone INXTA
```bash
git clone https://github.com/vertool/hx
```
5. Change Directory
```bash
cd hx
```
6. Add setup
```bash
chmod +x run.sh
```
7. Run the tool.
```bash
./run.sh
```
